# Emjay Brewery Frontend

A premium Next.js dashboard for juice manufacturing ERP.

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Lucide Icons
- Recharts
- Axios & React Hot Toast

## Features
- **Dashboard**: High-fidelity charts and inventory summaries.
- **Inventory**: Glass bottle and juice product tracking.
- **Production**: Batch filling with bottle stock validation.
- **Sales**: B2B (shops) and B2C (retail) order management.
- **Reports**: Financial auditing and data export UI.

## Local Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`

Environment variables:
`NEXT_PUBLIC_API_URL=http://localhost:5000/api`
